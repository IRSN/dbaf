DBAF is a database collecting the measurements of radiations performed by IRSN on Air-France flights. It has been designed by Paris Observatory and IRSN in the frame of the SIEVERT collaboration.
From this page you can perform the following tasks:

  • Upload some data concerning flights or measurements that respect DBAF data format.
  • Search for a one or more flights in the database, and get some informations about them, together with the measurements performed on these flights (plots, text).
  • Search for measurement devices and get the flights on which measurements have been performed.
  • Add some measurement devices, version numbers, conversion coefficient to dose (hawk)
  • Display the list of files, devices, flights which are stored in DBAF
To get an account, please contact DBAF manager, Francois Trompier (firstname.lastname@irsn.fr).
